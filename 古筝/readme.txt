改进：这个项目一开始是以简单，不吃配置为主，就算在cpu下运行也不难，所以并没有对手势识别和检测做太多工作，
在复杂场景下，识别精度会很糟糕。这里说一下改进思路，
①可以用移动检测判断背景(静止)和手势(移动)的差别，从而分离出手势来识别。
②可以用肤色检测来判断手的肤色，从而分离手势。
③用深度学习和大量样本训练，训练出更强的模型来，不过与我的初衷相悖。具体大家可以自行改进~
